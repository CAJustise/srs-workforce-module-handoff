# SRS Workforce Module Handoff

This repository is a clean handoff package to integrate the workforce features into another dashboard.

## Contents
- `workforce-module/README-Integration.md`
- `workforce-module/ROUTES.md`
- `workforce-module/PERMISSIONS.md`
- `workforce-module/TEST-CASES.md`
- `workforce-module/CLAUDE_PROMPT.md`
- `workforce-module/sql/workforce_module_schema.sql`

## Purpose
Use this package as implementation guidance for adding the following modules to an existing dashboard:
- Today
- Team Members
- Log Archive

## Notes
- This handoff package is intentionally separated from the original Spoonbill/SRS project repo.
- It can be shared with contractors or internal developers without granting access to your local machine.
